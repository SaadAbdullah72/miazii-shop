# MIAZI SHOP | Premium Tech & Electronics Emporium

A premium, production-ready MERN-stack eCommerce platform optimized for the high-end electronics market.

## ✨ Key Features
- **Premium UI/UX**: Professional, modern aesthetic with a focus on ease of use.
- **Mobile Responsive Excellence**: Tailored shopping experience on all devices, featuring a card-based mobile cart system.
- **Dynamic Pricing & Payments**:
    - **Digital Payment Incentive**: Digital payments are prioritized, while Cash on Delivery (COD) carries a 3% handling fee to manage logistics.
    - **Tax-Free Experience**: Global tax has been removed to simplify pricing for customers.
- **Fulfillment Manifest**: Advanced admin dashboard to track verified orders and mark them as delivered/fulfilled.
- **Order Management**: Sort, search, and delete orders directly from the administrative console.

## 🚀 Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, Lucide React
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Atlas)
- **Deployment**: Configured for Vercel Serverless

## 🛠️ Deployment Instructions
1. **GitHub**: Push to your repository.
2. **Vercel**: Import the project and add environment variables (`MONGO_URI`, `JWT_SECRET`, `EMAIL_PASS`, etc.).
3. **Configurations**: Ensure `BASE_URL` and `CLIENT_URL` point to your production domains.
